# data-vis-04

Advanced scatter plot example using `p5.chart.js` and p5.js v2.

## What it shows

- X: hour of day
- Y: dog age
- Point color: dog gender
- Point size: frequency

## Run

1. Open `/Users/jensykes/Documents/GitHub/dataVis/data-vis-04/index.html` with VS Code Live Server.
2. Confirm the scatter chart renders.

## Data

- `/Users/jensykes/Documents/GitHub/dataVis/data-vis-04/data/dog_spot.csv`
- Expected columns:
  - `Time Occured`
  - `Dog Age`
  - `Dog Gender`
  - `Frequency`
  - `Dog Breed`
